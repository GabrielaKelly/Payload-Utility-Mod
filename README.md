
## About
Payload is a 1.21.4 Minecraft Utility Mod made for Anarchy Servers such as 2b2t and Constantiam. Developed by deprecatedchase and pooseh.
The developers quit the project and released the source code to last public build to me and told me it was cool if I posted it. So here we are. The client is super unique and contains many modules that bypass on both 2b2t and Constantiam, and many unique modules such as FluctuatingTimer, AutoOminous, etc.

## Using the Jar
Download the Jar and move it into your existing mods folder. Make sure all dependency mods are present and there are no conflicting mods. The client is compatible with Future, Meteor, and RusherHack.

## Using the Source
Download the source and make sure your gradle is properly configured. Compile the client and the jar file can be found in libs\build.
